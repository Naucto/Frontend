# Naucto Frontend

---

## Prerequis

- [Node.js](https://nodejs.org/) should be installed

---

## setup

```bash
npm install
```
  
#### vscode workspace recommended for eslint
click to **File** then **Open workspace from file** and select the **settings/vscode/frontend.code-workspace** file

## Development

```bash
npm run dev
```

## Testing

```bash
npx jest            # unit tests
npm run cypress:run # e2e tests
```

## Docker

for dev, with hot reload feature
```bash
docker compose -f docker-compose.dev.yml up
```

for prod
```bash
docker compose up
```

---

It will then run on http://localhost:3001

## API

API to backend is autogenerated, please refer to the backend documentation for more details.
